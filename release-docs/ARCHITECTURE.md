## Project Architecture (folded)

**In scope (shipped shape):**
- Cover → Shift Desk → guide → play
- Training onboarding + four levels
- Classic win / loss reports; endless deepest-cleared scoring
- Multi-camera cave presentation with headlamp and blast feedback
- Local bests on hub cards

**Deferred:**
- Public production URL
- Endless milestones
- Dig aimed by camera look (possible later redesign)
- Cloud / account scores
- Click / touch dig, audio, multiplayer

**Presentation rule:** Expert field is described publicly as **30×16**.
